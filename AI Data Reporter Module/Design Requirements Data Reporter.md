AI API-powered data reporter to synthesize MBTA Red Line data from MBTA API and convert into useful insights for commuters.

App built from Alewife POV going into the city of Boston, destination Central Square in Cambridge.

Feed formatted data from the MBTA API to Ollama Cloud API to synthesize and return a report structured as noted below.

Report should read as a morning commute overview based on real-time information, including:

- Service alerts
- Next trains scheduled to leave Alewife in the upcoming hour
- On-time metrics
- Estimated time to destination

Total report length should not be longer than half a page. 

Output report as a .docx Word file
